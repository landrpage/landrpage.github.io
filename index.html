Website Templates

  • Choose from the list of available templates
  • Fill all the required details by looking at the instructions. These instructions help you understand which field appears where on the website
  • Checkout the live preview of your website and make changes if needed
  • If everything looks fine, create your website by making the payment

Code Access

  • Your website is basically a repository on Github
  • Your website code will be in HTML, CSS, JavaScript/jQuery
  • Once you get access to your website repository, it is as simple as cloning the code on your machine, making the changes and pushing it to see the changes on your website in minutes
  • If you're not a technical person and want us to make the desired changes on your website, we can take care of that for an additional price

Custom Domain

  • We will help you buy a domain of your choice
  • We will setup your website on the domain
  • We will teach you how to make changes to your website
